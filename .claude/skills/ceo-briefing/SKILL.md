# Skill: ceo-briefing

Generate the Monday Morning CEO Briefing — a weekly business audit that summarises revenue, completed tasks, bottlenecks, and proactive suggestions.

## Trigger
- `/ceo-briefing` — generate briefing for the current week
- `/ceo-briefing [YYYY-MM-DD]` — generate briefing for a specific week

---

## Hard Rules
- Auto-approved (no human approval needed) — this is a read-only report.
- Log generation to audit trail.
- Never write to Accounting/ or Odoo — read only.

---

## Step 1 — Collect Data

Read the following vault files:
1. `silver_tier/Business_Goals.md` — KPIs and targets
2. `silver_tier/Accounting/Current_Month.md` — revenue and expenses
3. `silver_tier/Done/` — tasks completed in the last 7 days (read all files)
4. `silver_tier/Logs/YYYY-MM-DD.json` — audit logs for last 7 days
5. `silver_tier/Plans/` — any open/unfinished plans
6. `silver_tier/Pending_Approval/` — items still awaiting action

Also call:
```
mcp__odoo__odoo_get_revenue_summary()
```
to get live Odoo accounting data (if Odoo is running).

---

## Step 2 — Analyse

Calculate:
- **Revenue this week** — from Accounting/Current_Month.md + Odoo
- **Revenue MTD** — vs monthly target
- **Tasks completed** — count files moved to Done/ this week
- **Bottlenecks** — plans older than 3 days still not done
- **Pending approvals** — count and list
- **Subscription audit** — flag any subscriptions not used in 30 days

---

## Step 3 — Generate Briefing

Write to `silver_tier/Briefings/[YYYY-MM-DD]_Monday_Briefing.md`:

```markdown
---
generated: [YYYY-MM-DDTHH:MM:SSZ]
period: [last Monday] to [yesterday]
---

# Monday Morning CEO Briefing

## Executive Summary
[2-3 sentence summary of the week — revenue vs target, key wins, any blockers]

## Revenue
- **This Week**: PKR [amount]
- **MTD**: PKR [amount] ([X]% of PKR [target] target)
- **Trend**: [On track / Behind / Ahead]
- **Invoices Pending**: [count] × PKR [total]

## Completed Tasks
[List tasks from Done/ this week with checkboxes]
- [x] [task name] — [date]

## Open Items
| Item | Age | Status |
|------|-----|--------|
| [plan name] | [N days] | [status] |

## Pending Approvals
| File | Action | Created |
|------|--------|---------|
| [filename] | [action type] | [date] |

## Bottlenecks
| Task | Expected | Actual | Delay |
|------|----------|--------|-------|
| [task] | [days] | [days] | +[N] days |

## Proactive Suggestions

### Cost Optimisation
[Flag unused subscriptions or high-cost items]

### Upcoming Deadlines
[List anything due in next 14 days]

### Growth Opportunities
[Based on Business_Goals.md — what should be prioritised this week]

---
*Generated by AI Employee — [timestamp]*
```

---

## Step 4 — Update Dashboard

Update `silver_tier/Dashboard.md` with a one-line briefing summary:
```
## Last CEO Briefing
[YYYY-MM-DD] — Revenue PKR [X] MTD ([Y]% of target). [N] tasks done. [N] pending approvals.
```

---

## Step 5 — Log & Confirm

Log to audit trail:
```python
logger.log_action("briefing_generated", "ceo_briefing", {"period": "..."}, "not_required", "n/a", "success")
```

Print:
```
=== CEO Briefing Generated ===
File:   silver_tier/Briefings/[filename]
Period: [date range]
Revenue MTD: PKR [X] / PKR [target]
Tasks done: [N] | Pending: [N]
==============================
```

---

## Scheduling

To run every Monday at 7:00 AM automatically:
```bash
python ceo_briefing_generator.py --schedule
```
This registers a Windows Task Scheduler job (or prints cron syntax on Linux/Mac).
