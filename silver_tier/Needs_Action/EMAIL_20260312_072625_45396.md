### Body (preview)

When using MCPServerStdio, Python&#39;s multiprocessing resource tracker can emit warnings about leaked semaphore objects at shutdown. This is a known limitation of the stdio transport and not a bug in user code.



This fix suppresses the warning at module import time using `warnings.filterwarnings()`, which prevents the confusing warning from appearing during program shutdown.
