The tool names need to be registered in advance when performing Responses API call (meaning sending input to the model you use). If you would like to load tools in demand, please consider the tool search feature we've recently added: https://github.com/openai/openai-agents-python/blob/main/examples/tools/tool_search.py



If you go with other models than OpenAI's Responses API compatible ones, there is no simple solution for your need at this moment.
